# Confirmaciones de colaboradores

En esta carpeta cada colaborador debe agregar su archivo de confirmación.

## Formato del archivo

```text
nombre-apellido.md
```

Ejemplo:

```text
juan-perez.md
```

## Importante

Cada colaborador debe crear su propio archivo.

No editar archivos de confirmación de otros colaboradores.

Usar como base la plantilla:

```text
plantilla-confirmacion.md
```
